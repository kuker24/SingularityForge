# Session Log

## Purpose

Write durable context without dumping chat history.

## Log Template

```md
# Session Log - YYYY-MM-DD

## Goal

## Files touched

## Commands run

## Decisions

## Verification

## Risks

## Next steps
```

## Rules

1. Keep logs factual.
2. Do not store secrets.
3. Mention failed commands honestly.
4. Prefer summaries over raw output.
5. Write next steps that can be resumed later.
